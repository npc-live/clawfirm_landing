# ClawFirm Landing Page

一人公司的终极武器 / Your AI Business Partner

## 🦞 简介

ClawFirm 是专为新时代个体创业者打造的"一人公司"自动化引擎。通过 AI 深度嵌入商业全链路，让一个人也能拥有一座工厂的战斗力。

## ✨ 特性

- 🌐 中英双语切换
- 📱 响应式设计
- 🎨 现代深色主题
- 💳 付费服务展示
- 📖 自助部署指南

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/OliviaPp8/clawfirm-landing.git

# 进入目录
cd clawfirm-landing

# 本地预览（任选一种）
python -m http.server 8080
# 或
npx serve .
```

然后访问 http://localhost:8080

## 📁 文件结构

```
clawfirm-landing/
```

## 🎨 自定义

1. **修改微信号**: 搜索 `ppgoodstuff` 替换为你的微信号
2. **添加头图**: 放入 `hero-image.png`
3. **添加二维码**: 放入 `wechat-qr.png`
4. **修改价格**: 搜索 `$299` 修改
