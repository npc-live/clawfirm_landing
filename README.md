## 🚀 快速开始 / Quick Start

```bash
# 克隆仓库 / Clone the repo
git clone https://github.com/npc-live/clawfirm_landing.git

# 进入目录 / Enter directory
cd clawfirm_landing

# 本地预览（任选一种）/ Local preview (pick one)
python -m http.server 8080
# 或 / or
npx serve .
```

然后访问 / Then visit http://localhost:8080
